extra_mods.run.build_front(args)[source]¶

Builds the frontends This whole process is dockerized so you don’t need any local nodejs installation ( but if wanted a local nodejs installation can be used also ) 1. Build the frontend docker image 2. Run the container ( keep it running artifically see Dockerfile.front ) 3. Run npm i 4. For all frontends ( check env.FR_FRONTENDS ) run npm i 5. For all frontends run webpack build 6. Kill the frontend container
